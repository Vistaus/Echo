## The Project

Echo is a simple utility to ping websites using GTK4 and Libadwaita.

## Features

Besides doing the default ping, you can configure the:
- Number of pings
- Interval between each packet
- Timeout
- And more

## Planned
- Ping history
- Stop ping midway
